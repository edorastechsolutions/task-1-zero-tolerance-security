## Why middleware?
Centralized enforcement prevents bypassing security logic.

## Brute force prevention
Generic responses stop username enumeration.

## Fail Closed
If any security module fails, request is denied.

## Logging Failure
System denies access to maintain security posture.

## Missing Configuration
Application exits during startup.

## Request Tracing
Unique request IDs allow forensic tracking.

## SIEM Integration
Logs are JSON structured and compatible with ELK/Splunk.
